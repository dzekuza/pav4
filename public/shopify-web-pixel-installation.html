🎯 Shopify Web Pixel Installation - Official API

🚀 RECOMMENDED APPROACH! This uses Shopify's official Web Pixels API for reliable checkout tracking.

This guide shows you how to install the ipick.io Web Pixel using Shopify's official Web Pixels API. This approach provides the most reliable tracking, especially for checkout events.

✅ Advantages of Web Pixels API:
  • Official Shopify API - No CSP restrictions
  • Reliable Checkout Tracking - Works on all checkout pages
  • Rich Event Data - Complete order and customer information
  • No Theme Modifications - Installed via Shopify Admin
  • Future-Proof - Uses Shopify's recommended approach

📋 Events Tracked

  • Page Views: Every page visit with detailed page information
  • Product Views: Product page interactions with complete product data
  • Cart Views: Cart page visits with item details
  • Add to Cart: Product additions with variant information
  • Checkout Started: Checkout initiation with cart details
  • Checkout Completed: Purchase completion with full order data
  • Checkout Steps: Address, shipping, and payment submissions
  • Collection Views: Collection page visits
  • Search: Search queries and results

🚀 Installation Methods

Method 1: Shopify Admin Web Pixels (Recommended)

  1. Go to your Shopify Admin → Settings → Apps and sales channels
  2. Click "Develop apps" → "Create an app"
  3. Name your app: "ipick.io Web Pixel"
  4. Go to "Extensions" → "Add extension" → "Web pixel"
  5. Configure the web pixel with the code below
  6. Install the web pixel on your store

Web Pixel Code:

// Copy this code into your web pixel extension import {register} from '@shopify/web-pixels-extension'; register(({analytics}) => { // Configuration const config = { apiUrl: "https://pavlo4.netlify.app/.netlify/functions/track-event", businessId: "2", // Replace with your business ID affiliateId: "aff_godislovel_1755091745057_n7ccoo", // Replace with your affiliate ID apiKey: "16272754ed68cbdcb55e8f579703d92e" }; // Extract business ID and affiliate ID from URL parameters function extractBusinessId() { const urlParams = new URLSearchParams(window.location.search); return urlParams.get('business_id') || urlParams.get('utm_source') || config.businessId; } function extractAffiliateId() { const urlParams = new URLSearchParams(window.location.search); return urlParams.get('affiliate_id') || urlParams.get('utm_medium') || config.affiliateId; } // Send tracking data to our API function sendTrackingData(eventType, data) { const businessId = extractBusinessId(); const affiliateId = extractAffiliateId(); const trackingData = { event_type: eventType, business_id: businessId, affiliate_id: affiliateId, platform: "shopify", session_id: `webpixel_${Date.now()}_${Math.random().toString(36).substr(2, 9)}`, user_agent: navigator.userAgent, referrer: document.referrer, timestamp: Date.now(), url: window.location.href, page_title: document.title, data: { ...data, shop_domain: window.location.hostname, source: "shopify-web-pixel" } }; // Send to our API fetch(config.apiUrl, { method: "POST", headers: { "Content-Type": "application/json", "Authorization": `Bearer ${config.apiKey}` }, body: JSON.stringify(trackingData), keepalive: true }) .then(response => { if (response.ok) { console.log(`[ipick Web Pixel] ${eventType} event sent successfully`); return response.json(); } else { throw new Error(`HTTP ${response.status}`); } }) .catch(error => { console.error(`[ipick Web Pixel] Failed to send ${eventType} event:`, error); }); } // Track checkout completed events (most important!) analytics.subscribe('checkout_completed', (event) => { console.log('[ipick Web Pixel] Checkout completed event:', event); const checkout = event.data.checkout; const orderData = { checkout_id: checkout.id, order_id: checkout.order?.id, total_price: checkout.totalPrice?.amount, currency: checkout.totalPrice?.currencyCode, subtotal_price: checkout.subtotalPrice?.amount, total_tax: checkout.totalTax?.amount, total_discounts: checkout.totalDiscounts?.amount, item_count: checkout.lineItems?.length || 0, email: checkout.email, phone: checkout.phone, customer_id: checkout.customer?.id }; // Extract line items if (checkout.lineItems) { orderData.items = checkout.lineItems.map(item => ({ product_id: item.product?.id, variant_id: item.variant?.id, title: item.title, quantity: item.quantity, price: item.cost?.totalAmount?.amount, discount: item.discountAllocations?.[0]?.amount })); } sendTrackingData('purchase_complete', orderData); }); // Track other events analytics.subscribe('page_viewed', (event) => { sendTrackingData('page_view', { page_title: event.data.page.title, page_url: event.data.page.url, page_type: event.data.page.type }); }); analytics.subscribe('product_viewed', (event) => { const product = event.data.product; sendTrackingData('product_view', { product_id: product.id, product_name: product.title, price: product.price?.amount, currency: product.price?.currencyCode }); }); analytics.subscribe('product_added_to_cart', (event) => { const product = event.data.product; const variant = event.data.variant; sendTrackingData('add_to_cart', { product_id: product.id, product_name: product.title, variant_id: variant.id, price: variant.price?.amount, quantity: event.data.quantity }); }); analytics.subscribe('checkout_started', (event) => { const checkout = event.data.checkout; sendTrackingData('checkout_start', { checkout_id: checkout.id, total_price: checkout.totalPrice?.amount, item_count: checkout.lineItems?.length || 0 }); }); console.log('[ipick Web Pixel] Web pixel registered successfully'); });

🔧 Configuration

Business ID & Affiliate ID Setup

The web pixel automatically detects your business ID and affiliate ID from:

  1. Hardcoded Config: Set in the web pixel code
  2. URL Parameters: ?business_id=2&affiliate_id=aff_xxx
  3. UTM Parameters: ?utm_source=2&utm_medium=aff_xxx
  4. Default Values: Falls back to configured defaults

🔍 Troubleshooting

Common Issues:

  • Web pixel not loading: Check app installation and permissions
  • No events tracked: Verify web pixel is active in Shopify Admin
  • Checkout not tracking: Web pixels work reliably on checkout pages
  • Wrong business ID: Check URL parameters or hardcoded config
  • API errors: Verify API key and endpoint URL
