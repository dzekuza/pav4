üöÄ Installation Methods

Method 1: Script Tag Installation (Recommended)

  1. Add the tracker script to your website's <head> section
  2. Configure your business ID and affiliate ID
  3. Test the installation

Basic Installation:

<!-- Add this to your website's <head> section --> <script src="https://ipick.io/tracker.js"></script> <meta name="ipick-business-id" content="YOUR_BUSINESS_ID"> <meta name="ipick-affiliate-id" content="YOUR_AFFILIATE_ID">

With URL Parameters:

<!-- For dynamic tracking with URL parameters --> <script src="https://ipick.io/tracker.js"></script> <!-- Business ID and affiliate ID will be extracted from URL parameters --> <!-- Example URL: https://yoursite.com/product?business_id=123&affiliate_id=aff_xxx -->

Method 2: Manual Integration

  1. Download the tracker.js file
  2. Host it on your server
  3. Include it in your website

Download and Host:

<!-- Download tracker.js and host it on your server --> <script src="/path/to/your/tracker.js"></script> <meta name="ipick-business-id" content="YOUR_BUSINESS_ID"> <meta name="ipick-affiliate-id" content="YOUR_AFFILIATE_ID">

üîß Configuration

Business ID & Affiliate ID Setup

The tracker automatically detects your business ID and affiliate ID from:

  1. Meta Tags: <meta name="ipick-business-id" content="123">
  2. URL Parameters: ?business_id=123&affiliate_id=aff_xxx
  3. Fallback Values: Configured defaults if not found

Example Configuration

<!DOCTYPE html> <html> <head> <title>Your Website</title> <!-- iPick Tracker Configuration --> <meta name="ipick-business-id" content="123"> <meta name="ipick-affiliate-id" content="aff_godislovel_1755091745057_n7ccoo"> <!-- iPick Tracker Script --> <script src="https://ipick.io/tracker.js"></script> </head> <body> <!-- Your website content --> </body> </html>

üîç Troubleshooting

Common Issues:

  • Tracker not loading: Check script URL and network connectivity
  • No events tracked: Verify business ID and affiliate ID are set
  • Wrong business ID: Check meta tags or URL parameters
  • API errors: Verify tracking endpoint is accessible
