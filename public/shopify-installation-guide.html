üöÄ PriceHunt Shopify Tracking Installation Guide

Follow this guide to properly install PriceHunt tracking on your Shopify store

Step 1: Access Your Shopify Theme

  1. Log in to your Shopify admin panel
  2. Go to Online Store ‚Üí Themes
  3. Click Actions ‚Üí Edit code on your active theme

Step 2: Add Head Script

In the theme editor, find and open theme.liquid file. Add the following code in the <head> section:

<!-- PriceHunt Tracking - Head Script -->
<script 
    src="https://ipick.io/shopify-tracker-head.js"
    data-business-id="YOUR_BUSINESS_ID"
    data-affiliate-id="YOUR_AFFILIATE_ID"
    data-debug="false">
</script>
Important: Replace YOUR_BUSINESS_ID and YOUR_AFFILIATE_ID with your actual PriceHunt credentials.

Step 3: Add Body Script

In the same theme.liquid file, add the following code just before the closing </body> tag:

<!-- PriceHunt Tracking - Body Script -->
        <script src="https://ipick.io/shopify-tracker-body.js"></script>

Step 4: Save and Test

  1. Click Save to save your changes
  2. Visit your store and test the tracking by:
    • Viewing a product page
    • Adding an item to cart
    • Starting checkout
    • Completing a purchase

Here's how your theme.liquid file should look:

‚úÖ What Gets Tracked

  • Page Views: Every page visit on your store
  • Product Views: When customers view product pages
  • Add to Cart: When items are added to cart
  • Checkout Start: When customers begin checkout
  • Purchase Completion: When orders are completed

‚ö†Ô∏è Troubleshooting

  • Events not showing: Check browser console for errors
  • Duplicate events: Ensure scripts are only added once
  • Purchase not tracked: Verify you're on the thank-you page
  • Debug mode: Set data-debug="true" for detailed logs

Step 5: Verify Installation

To verify the tracking is working:

  1. Open browser developer tools (F12)
  2. Go to the Console tab
  3. Look for [PriceHunt] log messages
  4. Check your PriceHunt dashboard for incoming events
